# Skill: Correo (ELIJA)

## Propósito
Revisar correo laboral, priorizar mensajes y preparar respuestas accionables con contexto de negocio.

## Cuándo usar este skill
- El usuario pide revisar su bandeja de entrada.
- Se requiere priorizar correos del día.
- Se necesita redactar respuestas rápidas o de alto impacto.

## Entradas esperadas
- Lista de correos nuevos/no leídos.
- Remitente, asunto, fecha y contenido clave.
- Contexto del usuario (proyectos, prioridades, compromisos).

## Flujo de trabajo
1. Clasificar correos por prioridad: `P1`, `P2`, `P3`.
2. Identificar correos que requieren respuesta hoy.
3. Detectar bloqueos, riesgos o decisiones pendientes.
4. Proponer borradores por prioridad.
5. Presentar plan de seguimiento (qué responder, delegar, agendar o archivar).

## Criterios de priorización
- `P1`: impacto alto y urgencia hoy.
- `P2`: impacto alto con ventana de respuesta esta semana.
- `P3`: informativo, seguimiento pasivo o archivo.

## Formato de salida recomendado
1. Resumen ejecutivo de bandeja.
2. Correos `P1` con acción recomendada.
3. Borradores sugeridos listos para enviar.
4. Pendientes de seguimiento con fecha objetivo.

## Criterios de calidad
- Respuestas claras, breves y orientadas a decisión.
- Confirmación de fechas y compromisos antes de enviar.
- Tono profesional alineado con la marca.
- Sin inventar datos no confirmados.

## Notas operativas
- Si falta contexto para responder, marcar como `[requiere contexto]`.
- Si el correo implica reunión, derivar al skill de agenda.
- Si el correo crea una tarea, derivar al skill de pendientes.
