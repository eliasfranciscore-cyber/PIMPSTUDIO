# Skill: Web Profesional (ELIJA)

## Proposito
Disenar y construir paginas web profesionales para la barberia, con foco en conversion, reservas y experiencia de usuario.

## Cuando usar este skill
- Se requiere crear o mejorar la web principal.
- Se necesita una landing de conversion.
- Se solicita diseno responsive con modulos funcionales.
- Se debe preparar estructura para escalar a app de reservas.

## Objetivos del skill
- Traducir necesidades del negocio a una arquitectura web clara.
- Crear interfaces modernas, rapidas y faciles de usar.
- Integrar modulos clave: servicios, reservas, cursos, contacto, mapa, newsletter y acceso de usuarios.

## Flujo de trabajo
1. Definir objetivo de la pagina y publico.
2. Estructurar secciones y flujo de navegacion.
3. Disenar UI (jerarquia visual, tipografia, color y componentes).
4. Implementar frontend responsive.
5. Conectar logica basica (formularios, reservas, cuenta usuario).
6. Validar usabilidad, rendimiento y consistencia.

## Estructura minima recomendada
1. Hero con CTA principal de reserva.
2. Servicios y precios.
3. Equipo de barberos.
4. Reserva de hora.
5. Cursos para nuevos barberos.
6. Contacto y mapa real.
7. Newsletter.
8. Acceso de usuarios / perfil.

## Criterios de calidad
- Diseno profesional y coherente con marca.
- Experiencia mobile-first y desktop robusta.
- Contenido claro orientado a accion.
- Formularios utiles y validaciones basicas.
- Codigo limpio, modular y mantenible.

## Entregables esperados
- Prototipo funcional en frontend.
- Estructura de componentes/paginas.
- Base de estilos reutilizable.
- Documentacion breve de siguientes pasos tecnicos.
