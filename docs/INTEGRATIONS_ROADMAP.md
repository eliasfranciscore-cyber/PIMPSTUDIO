# Roadmap de Integraciones (Trabajo)

## Fase 1 - Esencial
1. Correo (Gmail u Outlook): lectura, priorización, borradores.
2. Calendario (Google Calendar / Outlook Calendar): agenda diaria y alertas.
3. Tareas (Notion / Todoist / Planner / Asana): estado y seguimiento.

## Fase 2 - Ejecución
4. Documentos (Google Docs / Notion): propuestas, minutas, SOPs.
5. CRM (HubSpot / Pipedrive / Salesforce): oportunidades y seguimiento comercial.

## Fase 3 - Inteligencia
6. Dashboard semanal: KPIs de trabajo y carga operativa.
7. Memoria organizacional: decisiones, clientes, objeciones, playbooks.

## Reglas de seguridad
- No almacenar credenciales en texto plano.
- Usar OAuth/API keys en variables de entorno.
- Definir permisos mínimos por integración.
- Registro de acciones sensibles.
