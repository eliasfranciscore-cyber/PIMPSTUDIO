# Sistema Operativo de ELIJA

## Rutina diaria (L-V)

### 1) Brief de mañana (10-15 min)
Salida esperada:
- Top 3 prioridades del día.
- Riesgos de agenda (choques, reuniones sin objetivo, espacios de foco).
- Correos críticos pendientes de respuesta.
- Pendientes bloqueados y desbloqueo sugerido.

### 2) Bloque operativo (2-3 ciclos)
Cada ciclo:
- Revisar entrada (correo/tareas/mensajes).
- Clasificar: hacer, delegar, responder, programar, archivar.
- Ejecutar respuestas y seguimiento.

### 3) Cierre del día (10 min)
Salida esperada:
- Logros.
- Pendientes trasladados.
- Preparación del primer bloque del día siguiente.

## Matriz de prioridad
- P1: impacto alto + urgencia alta (hoy).
- P2: impacto alto + urgencia media (esta semana).
- P3: mantenimiento, documentación, mejoras.

## Formato estándar de reporte

### Resumen ejecutivo
2-5 líneas de estado.

### Acciones recomendadas
1. Acción + responsable + fecha.
2. Acción + responsable + fecha.
3. Acción + responsable + fecha.

### Riesgos
- Riesgo + impacto + mitigación.

### Decisiones pendientes
- Decisión + fecha límite + contexto mínimo.
